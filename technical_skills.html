Ikdienā turpinu pašmācības ceļā apgūt Python valodas prasmes. Galvenokārt ar Django tīmekļa ietvaru, apvienojumā ar Bootstrap front-end rīku. Neiztikt arī bez HTML, CSS un JS lietošanas, lai gan varu atzīties, ka pēdējais pieminētais galīgi nav mana tējas tase. Esmu liels Python piekritējs, gan valodas pārskatāmās sintakses dēļ, gan arī skaista 'Pythonic Style' koda dēļ.

Esmu ilglaicīgs Linux operētājsistēmas atbalstītājs un lietotājs. Jāsaka gan, ka pēc visiem interneta foruma likumiem esmu parasts plebejs, jo savā idkienā sanācis darboties tikai ar Ubuntu distributīvu.

Draudzējos ar Git kontroles sistēmu. Māku pamata lietas - noklonet repozitoriju, darboties ar to un pušot atpakaļ pamata atzarā. Lai vieglāk menedžēt savus personīgos kodus izmantoju GitHub piedāvātās iespējas. Neesmu gan tam piesiets un nepieciešamības gadījumā būtu gatabs izmantot kādu no alternatīvām (piemēram, GitLab vai Bitbucket)

Esmu nedaudz darbojies ar vienkāršotu C++ kodu un Arduino mikrokontrolieri. Konkrētāk Arduino Uno. Ir veikti vairāki maza apjoma darbi Tinkercad vidē, kā arī izmēģināts darboties ar dažādiem sensoriem un DC motoriem arī praktiskā veidā. Šobrīd nu jau kādu laiku mans Arduino Uno guļ atvilknē un gaida, kad man parādīsies iedvesma to atkal likt pie darba.

Adobe CC vide tiek izmantota gandrīz katru dienu nu jau 8. gadu no vietas. Darbā strādāju ar vektora failiem (Adobe Illustrator), veidoju maketus, produktu reprezentējošus attēlus, konceptus (Adobe Photoshop), savukārt brīvajā laikā fotogrāfiju apstrādē tiek pielietots Adobe Lightroom.
